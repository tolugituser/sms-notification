# QA Assessment

## Postman Collection

This repository contains a postman collection of functional tests for an sms microservice responsible for sending sms notifications.
The collection contains functional tests that can be imported and it is fully testable as it was create via a mockserver.

## Jmeter Concurrent Test File

This repository contains a jmeter load test simulating 50 concurrent requests to the send-sms endpoint.
This file can be imported and run on jmeter if you have it installed.
